MorphAgent

Typical values: 0 (desktop), 5 (mobile), 10 (tablet)
Blocks all JavaScript execution on websites. Pages will need to be reloaded after toggling.
Prevents websites from detecting spoofed user agent via JavaScript. Spoofs navigator properties, hides WebDriver flag, adds fake plugins, and protects against canvas/WebGL fingerprinting.
